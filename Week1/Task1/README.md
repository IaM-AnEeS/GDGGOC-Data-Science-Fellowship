# Programs Included

1. **arcs_of_circle.py**  
   Calculates the area and circumference of a circle.

2. **factorial.py**  
   Finds the factorial of a number.

3. **string_reverse.py**  
   Reverses a given string.

4. **number_guessing_game.py**  
   A simple guessing game using random numbers.
