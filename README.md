# AI/ML Fellowship – GDGOC COMSATS Attock

This repository contains all tasks, assignments, and projects completed during the AI/ML Fellowship Program organized by GDGOC at COMSATS University Attock.

#  Overview
The fellowship is designed to build practical skills in:

- Python programming
- Data analysis and visualization
- Machine learning fundamentals
- Advanced ML techniques
- Deep learning and computer vision
- End-to-end project development

# Program Breakdown

## Week 1 – Python Foundations & Git

### Task 1: Python Fundamentals

- Understanding variables, data types, and operators
- Control structures and loops
- Writing functions and basic problem-solving

### Task 2: Modular Python & Data Handling

- Advanced function usage (*args, **kwargs, lambda)
- Working with files and JSON data
- Exception handling (try/except/else/finally)
- Data structures and list/dict comprehensions
- Basics of algorithm efficiency

### Task 3: Object-Oriented & Advanced Concepts

- Core OOP principles (classes, inheritance, polymorphism)
- Encapsulation and special (magic) methods
- Creating custom decorators
- Iterators and generators
- Debugging strategies

### Task 4: Packaging & Production Readiness

- Writing modular code
- Managing virtual environments
- Building custom Python packages

### Task 5: Mini End-to-End Project

- Developing a real-world Python application
- Building a UI using Streamlit
- Planning, implementing, and deploying the project

## Week 2 – Data Handling & Visualization

### Task 6: Titanic Dataset EDA

Performing exploratory data analysis on the Titanic dataset

## Week 5 – Machine Learning Basics

### Task 7: Linear Regression

- Introduction to machine learning concepts
- Implementing linear regression from scratch:
- Normal Equation
- Gradient Descent
- Using Scikit-learn for implementation
- Evaluating models (MSE, RMSE, MAE, R²)
- Extending to multiple linear regression

### Task 8: Logistic Regression

- Basics of classification problems
- Mathematical intuition (sigmoid function, log-loss)
- Building logistic regression from scratch
- Using Scikit-learn implementation
- Performance metrics:
- Accuracy, Precision, Recall, F1-score, ROC-AUC
- Visualizing decision boundaries
### Task 9: Model Evaluation (Theory)

- Evaluation techniques for regression and classification
- Bias vs variance tradeoff
- Understanding underfitting and overfitting

### Task 10: Data Preprocessing & Feature Engineering

- Handling missing data (mean/median/mode imputation)
- Feature scaling techniques
- Encoding categorical features
- Using pipelines and column transformers

## Week 6 – Advanced Machine Learning

### Task 11: Advanced Models & Comparison
- Regularization techniques (Ridge, Lasso)
- Tree-based models (Decision Trees, Random Forest)
- Support Vector Machines (SVM)
- Ensemble techniques (Boosting, Stacking)
- Hyperparameter optimization
- Housing price prediction project
- Comparative analysis of ML models

## Week 7 – Unsupervised Learning

### Task 12: Clustering & Market Basket Analysis

- K-Means clustering (from scratch)
- Hierarchical clustering with dendrograms
- Dimensionality reduction (PCA, t-SNE)
- Association rule mining (Apriori algorithm)
- Market basket analysis project

## Week 8 – Neural Networks & Computer Vision

### Task 13: Deep Learning & CV

- Perceptron (single neuron model)
- Neural network fundamentals (layers, neurons)
- Backpropagation implementation
- Building ANN from scratch
- Data preprocessing for deep learning
- CNN concepts (convolution & pooling layers)
- Image classification (MNIST, CIFAR-10)
- Image augmentation techniques
- Transfer learning (MobileNetV2, ResNet50)
- Fine-tuning pretrained models
- Introduction to object detection and segmentation
